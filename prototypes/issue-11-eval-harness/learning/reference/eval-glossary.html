OpenMagic eval glossary

TermMeaning
ScenarioOne named behavior we want evidence about.
PerturbationThe difficult condition introduced to challenge that behavior.
EvidenceThe observations that prove or disprove the expected behavior.
GateA deterministic requirement that must pass before merging.
DiagnosticA measurement that informs comparison without independently failing the build.
SmokeA narrow check that a real external integration currently works.
BaselineThe inherited behavior measured for comparison, not the target architecture.
